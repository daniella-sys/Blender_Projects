Personal 3D modeling portfolio and repository for tracking progress in Blender.
# My 3D Modeling Journey 🚀

Welcome to my repository dedicated to learning 3D modeling and computer graphics. Here, I document my progress, starting from absolute zero, moving from understanding basic primitives to creating complex stylized assets.

## 🛠️ Software Used
* **Blender** (Current version: 5.1+)

## 🎯 Learning Approach
I follow a **three-step learning framework** for each model:
1. **Replication:** Following tutorials step-by-step to understand the tools.
2. **Memory Practice:** Recreating the same model from scratch without looking at the guide.
3. **Modification:** Changing the design, adding unique details, and customizing colors to make it my own.

## 📂 Project Gallery

### 1. Stylized Low-Poly Well
My very first 3D project! This model helped me understand viewport navigation, primitive shapes bashing (blocking), basic extrusions (`E`), and material assignment.

* **Techniques learned:** Edit mode manipulation, Inset (`I`), Extrude (`E`), basic color palettes (Z-fighting resolution).
